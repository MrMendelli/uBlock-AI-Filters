# About

This repository was created for me to share my personal custom [uBlock Origin](<https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/>) filters. I began creating these to block intrusive AI feature prompts and various other elements from a growing number of domains.

Ask all you want, no, I will not participate in the slop.

# Installation

From the uBlock settings, navigate to the `My filters` tab and paste or insert the contents of [`filters.txt`](filters.txt) as needed.
